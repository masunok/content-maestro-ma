# PRD: 콘텐츠 마에스트로

## 1. 개요
콘텐츠 마에스트로는 사용자가 입력한 주제나 제목을 기반으로 AI가 자동으로 블로그 콘텐츠와 SEO 최적화 팁을 제공하는 서비스입니다.
Freemium 기반의 크레딧 결제 시스템을 통해 무료 체험 후 유료로 확장할 수 있습니다.

## 2. 목표
개인 및 기업 사용자가 손쉽게 고품질의 블로그 콘텐츠를 생성할 수 있도록 지원.
SEO 친화적인 콘텐츠를 제작하여 블로그의 검색 노출 및 트래픽 증가를 돕는 것.
크레딧 기반 Freemium 모델로 서비스 진입 장벽을 낮추고, 유료 결제로 수익을 창출.

## 3. 주요 기능
#### 3.1 랜딩 페이지
##### 구성 요소
 - 헤더(Header): 로고, 메뉴(서비스 소개, 가격, 로그인/회원가입).
 - 히어로 섹션(Hero Section): 서비스 핵심 가치 제시, “시작하기” CTA 버튼.
 - Pricing 섹션
   - 크레딧 기반 과금 모델
     - 최초 가입 시 무료 크레딧 10회 제공.
     - 크레딧이 없을 경우 콘텐츠 생성 불가.
     - 유료 구매 옵션:
       - 50 크레딧 = 9,900원 (베이직)
         - 150 크레딧 = 24,900원 (프로)
         - 500 크레딧 = 79,900원 (엔터프라이즈)

   - 결제 수단: 토스페이먼츠V2

 - 푸터(Footer): 서비스(요금제, 대시보드), 지원(support@contentmaestro.com), 개인정보처리방침(이용약관, 개인정보처리방침, 쿠키 정책)

#### 3.2 대시보드
##### 로그인 후 접근 가능 (Supabase 인증 기반)
##### 주요 기능
 - 입력 폼: 블로그 제목/주제 입력.
 - AI 콘텐츠 생성:
   - OpenAI GPT-4o 모델 사용.
   - 블로그 제목 + 본문 자동 생성.
   - 생성된 콘텐츠 하단에 SEO 최적화 팁 제공.

 - 크레딧 차감: 콘텐츠 1회 생성 시 1 크레딧 차감.
 - 크레딧 잔여량 표시 및 추가 구매 버튼 제공.

## 4. 기술 스택
 - Frontend
   - Next.js 14 (App Router 기반)
   - Tailwind CSS
   - ShadCN/UI 컴포넌트
 - Backend / Database
   - Supabase (인증, PostgreSQL 데이터 저장)
 - 결제
   - 토스페이먼츠V2 (크레딧 구매)
 - AI
   - OpenAI API (GPT-4o 모델 사용)

## 5. 사용자 플로우
 - 사용자는 랜딩 페이지 접속 → 서비스 소개 확인.
 - 회원가입 시 무료 10 크레딧 지급.
 - 대시보드에서 블로그 주제 입력 → 콘텐츠 + SEO 팁 자동 생성.
 - 크레딧 부족 시 결제 안내 → 토스페이먼츠V2로 크레딧 구매.
 - 지속 사용 → 크레딧 차감 및 리필 과정을 반복.

## 6. 비즈니스 모델
 - Freemium → 유료 전환
 - 무료 체험 후 콘텐츠 제작이 필요할수록 크레딧 충전 → 수익 발생.
 - 대량 크레딧 구매자(기업/마케터)를 위한 할인 정책으로 장기적 유저 확보.

## 7. 향후 확장 기능 (로드맵)
 - 생성 콘텐츠의 직접 편집 및 저장 기능.
 - 다국어 블로그 콘텐츠 생성 지원.
 - SEO 점수 자동 분석 및 개선 가이드 제공.
 - 팀/기업 단위 계정 및 크레딧 공유 기능.
